Files

On this panel you can manage the files belonging to the plugin. There are four basic types of files.

Javascript-Files

These files can be used to add JavaScript Objects an functions to the abbozza! GUI in the browser. In these files new blocks can be defined.

Java-Files

These Files provides Java source code for classe which may be used by the plugin. There are four subtypes of Java-files.

Http Handler

The class provided by this type of Java-File has to be a subclass of PluginHandler. They are used to provide a Http handler to the abbozza! server.

If the plugin is activated, it may be contacted via the URL http://localhost:54242/abbozza/plugin/<pluginid>/.

MonitorPanel

The class provided by this type of Java-File has to be a subclass of PluginMonitorPanel. It provides an additional panel in the abbozza! Monitor. It receives messages with the given prefix.

MonitorListener

The class provided by this type of Java-File has to be a subclass of PluginMonitorListener. It provides an additional listener in the abbozza! Monitor. It receives messages with the given prefix.

Install Files

These files can be used to add libraries to the native sources of the system. They are copied under the name target into the build system right before compiling.
