General Information

Display Name

This field contains the name of the plugin that is displayed to users.

ID

The ID is a short identifier for the plugin. It is used in URLs to access contents and services provided by the plugin.

It may consist of a maximum of 32 letters and digits.

Description

The description is a short text, describing the functionality of the plugin.

Parent option

The plugin can be activated and deactivated using a specific option in the settings dialog. This field contains the name of the option to which the plugin option is added as child.

  • If left empty, the plugin option is added as a new root option.
  • advanced is the option activating the advance blocks.
  • devices is the option that activates the device blocks.
